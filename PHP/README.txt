# Instructions for use:

1. Install XAMPP (make sure to include apache and mysql. nothing else is necessary) https://sourceforge.net/projects/xampp/
2. Download files and place them in XAMPP's htdocs folder. Windows: c/xampp/htdocs/
3. Open XAMPP (instructions are weird for Apple)
4. Click "start" next to "Apache"
5. Navigate to localhost/yourfoldernamehere in browser
